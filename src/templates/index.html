PR-CYBR-MGMT-AGENT Dashboard

Welcome to the management dashboard for overseeing cybersecurity initiatives in Puerto Rico.

Overview

This dashboard provides an interface to interact with the PR-CYBR-MGMT-AGENT, allowing users to manage and monitor cybersecurity operations effectively.

Features

  • Interactive Map: Visualize and update cybersecurity operations across regions.
  • API Integration: Communicate with backend services for real-time data and command execution.
  • Authentication: Secure access to dashboard functionalities.
  • Data Retrieval: Fetch and display operational data and reports.

Backend Architecture

The backend is built using lightweight frameworks like Flask or FastAPI, providing RESTful APIs for seamless interaction with the agent's core functions.

  • Dockerized Environment: The backend runs within a Docker container for consistent deployment.
  • Service Integration: Defined in docker-compose.yml for easy setup and management.
  • Provisioning Script: Use scripts/setup_dashboard_backend.sh to set up and launch the backend service.
